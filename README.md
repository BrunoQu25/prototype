# Versión 1.0.0

Esta primer versión del prototipo tiene como objetivo la validación de:
- Proceso de alquiler de juegos
- Información y atajos disponibles en página de inicio
- Información disponible en sección de "Sobre el juego"
- Datos solicitados para completar una reserva
- Modalidad de navegación y exploración en la aplicación
- Diseño y estética elegida
